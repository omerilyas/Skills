# Installing a skill from this registry

## With the CLI (recommended for technical users)

```
npx skillhub add https://github.com/omerilyas/Skills.git --skill <skill-id>
```

`<skill-id>` is the folder name under `skills/`. The CLI fetches just
that folder, validates it, and installs it into your agent's directory.

## Without the CLI

Open the skill's page on the AI Skills Hub website, click **Download ZIP**,
and follow the per-agent install steps shown next to the button.
