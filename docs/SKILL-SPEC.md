# Skill Package Specification

The canonical spec lives in the platform repo at `docs/SKILL-SPEC.md`.
A copy is mirrored here so authors writing skills against this registry
can review the rules without the platform repo at hand.

The validator at `scripts/validate-skills.ts` enforces the same rules
on every PR.

> **Authoritative source:** the platform repo's `docs/SKILL-SPEC.md`.
> If they diverge, that doc wins.
