# Cisco Demo Builder

## Summary
Scaffold a Cisco product demo with talk track and config.

## What This Skill Does
- Generates a demo runbook for a chosen product.
- Includes a step-by-step talk track.
- Provides sample configuration and recovery commands.

## When to Use This Skill
You need a customer-ready demo and want a consistent talk track.

## Supported Agents
Claude Code, Cursor, VS Code, Windsurf.
