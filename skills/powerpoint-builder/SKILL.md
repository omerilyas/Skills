# PowerPoint Builder

## Summary
Turn structured outlines into branded PowerPoint decks.

## What This Skill Does
- Parses an outline you provide (sections, bullets, owners).
- Builds a title slide, agenda, content slides, and a closing slide.
- Applies the team template (fonts, colors, layout).
- Inserts speaker notes for each slide.

## When to Use This Skill
You have bullet-point notes or a meeting brief and need a presentable deck in minutes.

## Supported Agents
Claude Code, Cursor, VS Code.

## Installation
Use the install command shown on the skill page, or download the ZIP and follow the steps inside.
