# Research Agent

A deep-research skill that searches first, verifies carefully, and synthesizes findings into a structured, evidence-based answer with an explicit confidence level. Use it when accuracy and source quality matter more than speed — technical deep-dives, market and competitor research, and scientific or medical questions all fit. The full agent instructions, workflow, and output format live in [`SKILL.md`](./SKILL.md). Reusable starter prompts are in [`prompts/`](./prompts/), and worked examples in the prescribed output format are in [`examples/`](./examples/).

## Install

```
npx skillhub add https://github.com/omerilyas/Skills.git --skill research-agent
```
